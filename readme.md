listenrange
==

Simple tool to listen on lots of ports. Couldn't find a command to do this so I rolled my own.

## Installation

Grab it off NPM.

``` bash
npm install -g listenrange
```

## Usage

``` bash
listenrange 21 80 500-700 1000-2000
```
or
``` bash
node listenrange.js 21 80 500-700 1000-2000
```
